Exam Themes

הנושאים למבחן

-- Arithmetics like: m = m * 5, n++, d**2, and for the strings - concatenation and template string
-- if-else constructs, comparisons. Ternary operator.
-- Using class interface classList with it's methods: add, remove, replace, contains (no toggle).
-- Getting random member of an array using the formula Math.floor(Math.random * arr.length).
-- Creating usual and arrow functions. Hoisted and not hoisted.
-- Calling functions from other functions.
-- Returning value from funciton.
-- Going with FOR or forEach loop over all the members of the array.
-- Using the same index for the parallel arrays of the same length.
-- Adding and removing elments from the array, using methods push(),pop(),shift(),unshift().
-- document.querySelector() (or document.getElementById()) and document.body.
-- Global and local variables, scope.
-- Inputs. Changing Data Type to numeric if needed.
-- Using toLowerCase() or toUpperCase() to simplify comparisons.
-- Adding event listener to HTML elements for the events "click","input","change","mouseover"
-- Using existing named functions as callback functions and creating anonymous callback functions.
-- Changing element content by using it's innerText or innerHTML property
-- All the steps of creating new element: document.createElement(), setting the new element's properties, like "src", id, class, innerText, etc., appending the new element to the parent element
-- Object and properties. Using property of an object that is an array member
-- includes() for strings and for arrays

-- פעולות חשבון כגון m = m * 5, n++, d**2, ועבור המחרוזות - concatenation (+) ו-template string (backticks)
-- מבנים של if-else , השוואות, האופרטור המשולש
-- שימוש בממשק הקלאסים classList והמתודות שלו: add, remove, replace, contains (הפעם ללא toggle) -- קבלת איבר רנדומאלי מתוך מערך בשימוש בנוסחא Math.floor(Math.random * arr.length).
-- יצירת פונקציות רגילות ופונקציות חץ (סדר כתיבה נכון, hoisting)
-- קריאה לפונקציות מתוך פונקציות אחרות.
-- החזרת ערך מתוך פונקציה.
-- מעבר על כל איברי המערך בעזרת לולאת FOR או forEach.
-- שימוש באותו האינדקס עבור מערך מקביל בעל אותו האורך.
-- הוספה והוצאת איברים ממערך כאשר משתמשים במתודות push(),pop(),shift(),unshift().
-- document.querySelector() (או document.getElementById()) ו- document.body.
-- משתנים גלובאליים, לוקאליים (מקומיים), scope
-- Inputs. קלט. שינוי של סוג נתון (data type) אם צריך.
-- שימוש ב-toLowerCase() או toUpperCase() על מנת להקל על ההשוואות.
-- יצירת ליסטנרים עבור האלמנטים של HTML עבור האירועים "click","input","change","mouseover".
-- שימוש בפונקציות קיימות בתור פונקציות callback ויצירת פונקציות callback אנונימיות.
-- שינוי תוכן של האלמנט בשימוש בתכונות innerText או innerHTML.
-- כל השלבים של יצירת אלמנט: document.createElement(), הקנאת תכונות חדשות כגון "src", id, class, innerText, וצירוף לאלמנט ההורה.
-- אובייקט ותכונותיו (properties). פניה לתכונה של אובייקט כאשר הוא באותו זמן גם הינו איבר של מערך.
-- includes() עבור מערכים ומחרוזות

-- Clarification: during the test it's OK to ask the teacher questions to understand the missions, or ask for technical help, but is not OK to ask how to solve the missions.

-- אל תשנו לא את HTML ולא את CSS, צריך רק למלא את המשימות כאן ב JS.
-- שימו לב שלכל משימה תמיד יש כמה אפשרויות לפתרון.
-- הבהרה: בזמן המבחן זה בסדר לשאול את המורה כדי להבהיר את המשימות או לבקש עזרה טכנית, אמנם זה לא בסדר לבקש עזרה בפתרון.
